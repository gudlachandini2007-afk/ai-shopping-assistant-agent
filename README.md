# AI Shopping Assistant Agent

AI-powered shopping assistant built using n8n, Google Gemini, Telegram, and workflow automation.

## Features

- Conversational AI shopping assistant
- Telegram bot integration
- AI-generated recommendations
- Workflow automation using n8n
- Agentic AI workflow design
- Prompt engineering based responses

## Tech Stack

- n8n
- Google Gemini API
- Telegram Bot API
- Generative AI
- Prompt Engineering

## Workflow Overview

The workflow receives user queries from Telegram, processes them using Google Gemini AI, and returns intelligent shopping recommendations automatically.

## Use Cases

- Product recommendations
- Fashion suggestions
- Smart shopping assistance
- AI automation workflows

## Future Improvements

- Amazon API integration

- Personalized recommendations
- Voice assistant support
